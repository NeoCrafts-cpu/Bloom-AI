# Bloom AI

**Agentic Financial Media & Execution Network (AFMEN)** — SoSoValue market intelligence and SoDEX live market data, routed through a five-agent pipeline from macro signals to risk-gated on-chain copy trading.

Built for the [SoSoValue 2026 Buildathon](https://sosovalue.com).

## What Bloom AI Does

Bloom AI turns institutional-grade market data into actionable output:

1. **Ingest** — SoSoValue ETF flows, news, sentiment, fundraising + SoDEX tickers, klines, orderbook
2. **Analyze** — Journalist and Chart Analyst agents publish newsletters and TA briefings
3. **Signal** — Strategist converts narrative → pipeline-generated SSI index weights with live SoDEX prices
4. **Protect** — Sentinel runs 8 deterministic risk checks before any trade
5. **Execute** — Broker submits EIP-712 signed batch orders to SoDEX testnet **only after user confirmation**

## Core Flow

```
SoSoValue + SoDEX
    → Journalist → Chart Analyst → Strategist → Sentinel preview
    → User picks strategy → Wallet sign → Sentinel check → Broker → SoDEX testnet
    → Verified session performance (no fabricated KPIs)
```

**Important:** Run **Run Full Pipeline** on the dashboard before visiting `/strategies` or `/copy-trade`. Strategies are generated by the pipeline — there is no static seed catalog.

## Target Users

Retail and crypto-native traders who want institutional-style market intelligence with explicit risk controls and one-click copy trading on ValueChain testnet.

### Agents

| Agent | Role | Trigger |
|-------|------|---------|
| **Journalist** | SoSoValue ETF flows, news, sentiment → Smart Money newsletter | Every 10 min + manual |
| **Chart Analyst** | SoDEX klines → RSI/SMA → TA briefing (LLM or deterministic fallback) | Every 2 h + manual |
| **Strategist** | Newsletter narrative → SSI index with live weights | Pipeline / post-newsletter |
| **Sentinel** | 8 deterministic risk rules — blocks bad trades | Copy-trade + pipeline preview |
| **Broker** | Multi-asset SoDEX batch orders | User confirmation only |

### Data Sources

| Source | Used For |
|--------|----------|
| **SoSoValue API** | ETF flows, news, sentiment, fundraising, currency data |
| **SoDEX Testnet** | Prices (primary), OHLCV klines, orderbook, account state, execution |
| **CoinGecko** | Price fallback when SoDEX unavailable |
| **DefiLlama** | DeFi TVL leaderboard |
| **CryptoPanic** | News headlines (optional) |
| **OpenRouter** | LLM newsletters and TA (optional — template fallback if missing) |

## Environment Variables

### API (`apps/api/.env`)

| Variable | Required | Description |
|----------|----------|-------------|
| `SOSOVALUE_API_KEY` | Yes (for full demo) | SoSoValue Terminal API key |
| `SODEX_SPOT_URL` | No | Default: `https://testnet-gw.sodex.dev/api/v1/spot` |
| `SODEX_API_KEY_NAME` | Live execution | Registered SoDEX API key name sent in `X-API-Key` |
| `SODEX_API_PRIVATE_KEY` | Live execution | EIP-712 signing key for SoDEX |
| `SODEX_API_KEY_ADDRESS` | Optional | SoDEX API key public address for nonce namespacing; derived from private key if omitted |
| `OPENROUTER_API_KEY` | Optional | LLM for newsletters/TA |
| `OPENROUTER_MODEL` | No | Default: `~anthropic/claude-sonnet-latest` |
| `WEB_URL` | Production | Frontend URL for CORS |
| `PORT` | No | Default 4000 |

### Web (`apps/web/.env.local`)

| Variable | Description |
|----------|-------------|
| `NEXT_PUBLIC_API_URL` | API URL (empty = same-origin proxy via Vercel rewrites) |
